a closer look

What do I know about Ruby Hashes?

So far, a few things. One, it's an unique way of storing data with key-value pairs. Two, the indexing of the hash is unlike that of the array. Every key becomes the source that one uses to access its value pair. The example below is a hash.

How does one create a hash?

It's a good question to ask. There are a few ways depending on one's purpose. You can create an empty hash or a hash with a default value. Both of these can be achieved with the new class method.

Do we have hashes in Javascript?

No, however, JS has Object Literals. JS Object Literals look similar to Ruby hashes. It has property name that pairs with values. When one want to get the key, the object has to be called. The examples below shows a side by side comparison of how hashes to objects.Code Embedding Source 

### creating new Ruby Hash literal
acctGroup = {
  person1: "Victoria",
  person2: "Robin",
  person3: "Shin",
  person4: "Chris",
  person5: "Buck"
}

### accessing Ruby hash value
p acctGroup[:person2]
# :> "Robin"
p acctGroup[:person4]
# :> "Chris"
p acctGroup[:teacher]
# :> nil

### changing Ruby hash value
acctGroup[:person3] = "Batman"

### adding a new Ruby key/value pair
acctGroup[:teacher] = "Emmanuel"

### deleting a Ruby key/value pair
acctGroup.delete(:person3)

### print out
acctGroup.each {|x| p x}
=begin
:>
[:person1, "Victoria"]
[:person2, "Robin"]
[:person4, "Chris"]
[:person5, "Buck"]
[:teacher, "Emmanuel"]
=end

// creating new JS object literal
var acctGroup = {
  person1: "Victoria",
  person2: "Robin",
  person3: "Shin",
  person4: "Chris",
  person5: "Buck"
};

// accessing JS value from objects
console.log(acctGroup["person2"]);
// > Robin
console.log(acctGroup["person4"]);
// > Chris
p acctGroup[:teacher];
// > undefined

// changing JS object value given the property
acctGroup["person3"] = "Superman";

// adding a new property name and value to object
acctGroup["teacher"] = "Emmanuel";

// deleting a property
delete acctGroup["person3"];

// print out
console.log(acctGroup);
/*
{ person1: 'Victoria',
  person2: 'Robin',
  person4: 'Chris',
  person5: 'Buck',
  teacher: 'Emmanuel'
}

Version Control: git and GitHub

Feb 14, 2016

What are the benefits of version control?

1. It allows collaboration between individuals of a team since different versions of the same file will eventually be merged. 2. It keeps track of the differences between files saved over time, so that one can revert to previous versions if necessary. 3. It allow a master to stay intact without fear of losing it.

How does git help you keep track of changes?

1. Git allows the user to provide detail comments on their commits while working in branches other than the master. Agreement to this workflow between individuals of a team creates an assurance that people are working towards a common goal.

Why use GitHub to store your code?

Things can happen (destroy) to personal and work computer harddrives. Having a remote repository provides the redundancy in the work that's done.
